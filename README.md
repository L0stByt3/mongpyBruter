# mongPyBruter
***
Sencilla herramienta para realizar ataques de diccionario a servidores de bases de datos Mongo
## Instalación
***
Instrucciones de Instalación.

git clone https://github.com/L0stByt3/mongPyBruter.git
## Instalar dependencias
- pip install tqdm pymongo
- cd mongPyBruter

## Forma de uso
python mongPyBruter.py -h host.txt -u users.txt -p passwords.txt

## Salida
mongo_user_pass_result.txt
